This plugin provides beautiful social sharing buttons in posts, pages and home page to display above, below the content and floating buttons.

== Description ==

Pingu Sharing Buttons is a plugin created by <a href="http://www.techpingu.com/">TechPingu</a> for all WordPress websites and blogs. Show beautiful buttons of Facebook, twitter, google plus, pinterest, stumble upon and LinkedIn. 

Main features of this plugin are:
<ul>
<li>Social sharing buttons of Facebook, Twitter, LinkedIn, Google Plus, Pinterest, StumbleUpon.</li>
<li>Show Buttons in Pages, Posts and Front Page.</li>
<li>Add Twitter username automatically while sharing the post.</li>
<li>Above the content, below the content and floating buttons in sidebar.</li>
<li>Responsive social sharing buttons.</li>
<li>Automatic Counting on total shares.</li>
<li>Hide and Show the Sharing Buttons from Backend.</li>
</ul>

For the premium version of this plugin you will have to buy it from <a href="http://products.techpingu.com/product/pingu-share-button/">here</a>.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Pingu Sharing Buttons` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Buy the premium version of plugin from http://products.techpingu.com/product/pingu-share-button/ 
5. Configure the plugin by going to the Pingu Sharing Buttons Menu

== Frequently Asked Questions ==

= What is the purpose of this Plugin? =

The plugin gives enormous simplicity to social sharing in websites. Smart icons, easy backend settings to show and hide the buttons, standard button styling. Above the content, below the content and floating responsive sharing buttons of Facebook, Twitter, LinkedIn, Google Plus, Pinterest, StumbleUpon.
